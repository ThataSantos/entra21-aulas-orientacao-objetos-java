# Aprendendo sobre Herança

## Herança é a abstração para aproveitar atributos e métodos de outra classe

> Essa classe que está provendo atributos e métodos, podemos chamar de super-classe
> Para realizar essa herança em uma subclasse temos que usar a palavra reservada **extends**
> Quando uma subclasse herda atributos de métodos da super-classe não é mais necessário manter meus atributos e métodos escritos na minha sub-classe
